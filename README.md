# Windows Server AD DS, DNS & DHCP Lab

## Project Overview
This lab project demonstrates the setup and basic configuration of a Windows Server environment including:

- Active Directory Domain Services (AD DS)
- DNS configuration
- DHCP configuration
- User and Organizational Unit management
- Basic administrative tasks

## Objectives
- Understand the role of Windows Server in an enterprise environment
- Practice installing and configuring AD DS
- Configure DNS and DHCP services
- Create and manage users and OUs
- Improve hands-on skills in system administration

## Environment
- Windows Server
- Virtual machine / lab environment
- Basic network setup

## Tasks Completed
- Installed Windows Server
- Promoted server to Domain Controller
- Configured Active Directory
- Set up DNS service
- Configured DHCP scope
- Created test users and organizational units
